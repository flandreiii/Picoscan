// get started

Up & Running
in Seconds

Clone the repository, install dependencies and run. That's genuinely it — Picoscan is designed to get out of your way so you can focus on what matters.

No complex configuration. No account required. No license key. Just open-source software that respects your time.

Star on GitHub ★ Download ZIP
terminal 
# Clone the repo
$ git clone https://github.com/flandreiii/Picoscan.git

# Move into directory
$ cd Picoscan

# Install dependencies
$ pip install -r requirements.txt

# Run Picoscan
$ python picoscan.py

# ✓ Ready to scan.
